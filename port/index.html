  Howdy, I'm Peter Muniz - a current IT professional and hopeful front-end (eventually full-stack) web developer. I dabbled in coding for years now and decided to make 2017 the year I land a job as a developer. I started the 100-days-of-code challenge and am near completion. During this time, I earned a front-end web certification from FreeCodeCamp which I am quite proud of. I'm a dedicated, passionate worker and lifetime learner. Hopefully I can bring my skillset to your company soon.

What I've Done

Simon

My rendition of the 80's classic Simon game. This was a very fun and challenging project for me. It was also the first time I drew real-life objects with CSS which was quite fun.

TicTacToe

My first "real" game. Deciding how the computer would know which moves had already been made was quite challenging. I also wanted to keep it interesting and not make the same choices. I think the end-result came out nicely.

Pomodoro Clock

An app I still use today to make sure I get my coding time in. Here, the user can define work and break time lengths and then receive an alert when those limits have been reached. I've always liked this font and finally found a place to use it.

Twitch

My third adventure into the world of JSON. I personally love Twitch and was excited to get the chance to create this friends-list type app. It displays a list of streamers, their status, and what game they're playing. I also added a feature where new streamers can be added to your watch-list.

Wikipedia Viewer

A compact wiki viewer that instantly displays a list of results. I fully realized the power of APIs creating this app. I had no idea how open-source the web really was until this project.

Weather

This little app takes your zipcode and returns the weather in your area. At the time, CodePen didn't play well with geolocation since it wasn't HTTPS compliant so I had to default back to manually entering the user's zip. This was my very first time working with JSON and APIs. While it was quite challenging, it ended up being a very rewarding experience.

Random Quote Machine

One of my first projects, this simple quote generator delights the user with various quotes from leaders in the technology field.
